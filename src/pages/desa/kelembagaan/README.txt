Pages:
- KelembagaanDesaPage: entry dashboard with cards for RW, RT, Posyandu, Karang Taruna, LPM, PKK.
- KelembagaanList: list for RW/RT/Posyandu.
- KelembagaanDetail: dynamic component for create/update and detail view for singleton kelembagaan; also RW actions include quick add RT.
- RwDetailPage: wrapper using KelembagaanDetail; can be extended to filter RT by RW id.

Services available in src/services/kelembagaan.js.
